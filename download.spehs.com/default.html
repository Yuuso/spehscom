The game is currently in a pre-alpha state. It contains a glimpse into most of the features that we wanted to develop for this game. Many of the game’s features are still in very early stages. The game balance is almost non-existent, but we think that its still fun to play. It does require dedication to get to know the controls and the “unique” user interface. The future of Spehs is still hazy, but the development will most likely continue some time in the future.
